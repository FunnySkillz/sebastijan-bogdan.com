# Storyboard: 5-Act Cinematic Stage

## Act I — The Machine Awakens (0.00-0.16)
- System Core is semi-exploded and calm.
- Identity overlay is dominant.
- Halo/core settle and label logic is readable.

## Transition I -> II (0.16-0.28)
- Act I overlay retracts quickly.
- `trans-12` geometry appears (plate/rail conversion).
- Stage rotates/shifts minimally while center anchor remains fixed.

## Act II — The System Opens (0.28-0.44)
- Experience overlay is dominant.
- Internal spine-style mode layer is emphasized.
- Node/spec narrative reads as architecture reveal.

## Transition II -> III (0.44-0.58)
- Experience mode compresses.
- `trans-23` routes appear.
- Cartridge geometry starts separating toward subsystem layout.

## Act III — Systems in Action (0.58-0.74)
- Project overlay is dominant.
- Act III panel emphasis shifts FleetFuel -> SteuerFux -> Automation.
- Shared core remains one machine while subsystem emphasis rotates.

## Transition III -> IV (0.74-0.84)
- `trans-34` simplifies modular explosion into grouped board language.
- Project mode recedes before toolbox overlay dominates.

## Act IV — Engineering Stack (0.84-0.93)
- Toolbox overlay is dominant.
- System reads as grouped capability map.
- Motion intensity is lower than Act III.

## Transition IV -> V + Resolve (0.93-1.00)
- `trans-45` collapses complexity.
- Philosophy/contact overlay becomes dominant.
- End state is minimal, stable, and resolved.

## Motion Intensity Curve
- Act I calm -> I->II reveal -> Act II controlled
- II->III strongest handoff -> Act III richest
- III->IV simplify -> Act IV measured
- IV->V reduce -> Act V restrained
