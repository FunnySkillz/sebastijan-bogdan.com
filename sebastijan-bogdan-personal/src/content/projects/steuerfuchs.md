## Current Status

SteuerFuchs is still a working title, and the branding is still evolving. Right now, the focus is making the beta genuinely useful: stable core flows, clear EN/DE language, and reliable export.

## Stack Snapshot

The app is built with Expo/React Native and TypeScript, with on-device SQLite for a local-first approach. The architecture is intentionally simple so I can ship quickly without creating long-term chaos.

## Challenges Along The Way

The biggest friction points were bilingual terminology consistency, export edge cases, and keeping code quality high while shipping quickly.

## Lessons Learned

Shipping early was the right decision because real user feedback surfaced priorities faster than planning alone. AI tooling helped speed up drafts and refactors, but domain-critical decisions still required strict human review.
